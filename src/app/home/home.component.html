Frequently Asked Questions
 How do I create a profile?

To create your profile, install the Osrs Profile plugin from the plugin hub on RuneLite. Your profile will be synced to our API every minute.

How do I remove my profile?

Please note that you should uninstall the Osrs Profile plugin from RuneLite before requesting the deletion of your profile. If the plugin sends your data to our API after your profile has been removed, your profile will just be created again automatically.
